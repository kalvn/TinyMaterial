Tiny Material CSS

This CSS toolbox embeds a few tools to facilitate the creation of a Material Design theme.
It's not a full CSS-framework like Bootstrap can be for example, so its purpose it not to replace those.

Some of the tools (color system, checkboxes, radio buttons and switches) are taken directly from the Materialize project.

Popups snippet
<button class="btn-raised" type="button" data-popup="#amazing-popup">Click on me!</button>
<div id="amazing-popup" class="popup">
    <div class="popup-header">This is my popup's content.</div>
    <div class="popup-body">
        <div class="popup-content">You can put some content.<br>Or even a list!</div>

Modals

Modals are elements which appear on top of everything else.

Don't forget to give them a width so they can be centered. modal-inner class is used to allow to give a width and keep the modal centered. If you avoid it, the modal will be 100% width.

Modal header
This is my body.
Modals snippet
<button class="btn-raised btn-primary mrm" type="button" data-modal="#amazing-modal">Open a modal window</button>
    <div id="amazing-modal" class="modal">
        <div class="modal-inner" style="width:500px">
            <div class="modal-header">Modal header</div>
            <div class="modal-body">This is my body.</div>
            <div class="modal-footer">
                <button class="btn ripple-primary" type="button" data-close="#amazing-modal">Save</button>
                <button class="btn" type="button" data-close="#amazing-modal">Cancel</button>
            </div>
        </div>
    </div>
